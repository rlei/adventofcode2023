## Advent of Code 2023 - Day 14

Written in [Crystal](https://crystal-lang.org/) `1.10.1`.

### Problem 1

To run:

`crystal run prob1.cr < path/to/input_data`

Or, build it first with `crystal build --release prob1.cr` then run `./prob1 < ...`.
